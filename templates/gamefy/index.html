About GameFy

GameFy is a free Web apps for the gamers that allows them to play online logic games.


GameFy features:

  1. Gamer registration for personal account.
  2. Playing amazing different online logic games.
  3. Personal dashboard.
  4. Updated scoreboard to compare brain speed.

Used technologies:

  • Django & Python
  • Javascript
  • Bootstrap
